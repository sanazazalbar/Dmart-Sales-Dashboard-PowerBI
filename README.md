# 🛒 D-Mart Sales Analysis Dashboard (Power BI)

## 📌 Project Overview
Built an interactive Power BI dashboard — **D-Mart Analysis** — 
to analyze retail sales performance across cities, product 
categories, and customer segments using DAX measures and 
data visualization.

## 🛠 Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation
- Data Visualization

## 📊 Key Metrics Tracked
| Metric | Description |
|---|---|
| Total Sales | Overall revenue generated |
| Total Profit | Net profit across all transactions |
| Total Quantity | Units sold across all categories |
| Profit Margin | Profitability ratio by category/city |
| Sales Amount | Transaction-level sales tracking |

## 🔍 Key Analyses Performed
- **Category-wise sales funnel** across 4 categories:
  Clothing, Electronics, Home Care & Grocery
- **Top 10 Cities by Total Sales** — geographic 
  revenue ranking
- **Monthly sales trend** using Order Date hierarchy
- **Customer Type analysis** — Regular vs Member 
  customer segmentation
- **Profit Margin** analysis by category and city

## 🌍 Dimensions Analyzed
- 4 Product Categories (Clothing, Electronics, 
  Home Care, Grocery)
- City-level geographic breakdown (Top 10 cities)
- Customer Type segmentation
- Monthly time-series trends

## 💡 Business Use Case
Retail businesses can use this dashboard to:
- Identify best and worst performing product categories
- Focus expansion in high-revenue cities
- Understand Regular vs Member customer buying behavior
- Track monthly revenue and profit trends

## 🚀 Conclusion
This dashboard provides a 360° view of D-Mart's retail 
performance, enabling data-driven decisions around 
inventory, pricing, and regional strategy.
